Web 动画系统概览

动画系统是 UI 交互的灵魂。在 Web 开发中，它主要由三个核心部分组成：

1. 关键帧 (Keyframes)

定义动画在特定时间点的状态。例如 0% 是开始，100% 是结束。

2. 补间/缓动 (Easing)

决定状态变化的“节奏”。是匀速运动，还是像弹簧一样先快后慢？

3. 属性转换 (Transforms)

操作元素的位移、旋转、缩放。通过 GPU 加速，确保动画在高负载下依然流畅。

4. 触发机制 (Triggers)

动画何时开始？可以是页面加载自启动，也可以是用户点击或悬停时触发。

技术小贴士

正如你在论文中看到的 TFPS 架构 使用多专家协作（MoE）来处理不同模式，现代动画系统也会根据不同的交互场景（模式）选择不同的渲染策略，以达到性能与视觉的最佳平衡。
